== Description ==  
ROI Investment Calculator WP is a lightweight and easy-to-use WordPress plugin that allows users to calculate their **Return on Investment (ROI)** instantly. This plugin provides a customizable investment calculator that can be displayed using a shortcode or as an **Elementor widget**.  

Plugin Documentation: [https://wordpress.org/plugins/roi-investment-calculator-wp/](https://wordpress.org/plugins/roi-investment-calculator-wp/)

== ⚡ Features ==  
* Simple and easy-to-use ROI calculator  
* Use the shortcode `[investment_calculator_wp]` to display the calculator  
* Elementor widget included for easy drag-and-drop usage  
* Fully responsive design for all devices  
* Customizable styles and settings  

== Installation ==  

= Minimum Requirements =  
* WordPress 5.8 or greater  
* PHP version 7.2 or greater  

= AUTOMATIC INSTALLATION =  
1. Log in to your WordPress dashboard.  
2. Navigate to the **Plugins** menu and click **Add New**.  
3. In the search field, type "**ROI Investment Calculator WP**" and click **Search Plugins**.  
4. Once found, click **Install Now** and then **Activate**.  

= MANUAL INSTALLATION =  
1. Download the plugin from [https://wordpress.org/plugins/roi-investment-calculator-wp/](https://wordpress.org/plugins/roi-investment-calculator-wp/).  
2. Upload `roi-investment-calculator-wp` to the `/wp-content/plugins/` directory.  
3. Activate the plugin through the 'Plugins' menu in WordPress.  

== How to Use ==  
1. After activation, go to the **Investment Calculator** menu in the WordPress dashboard.  
2. Copy and paste the shortcode `[investment_calculator_wp]` into any page or post.  
3. If using Elementor, find the **ROI Investment Calculator** widget and drag it to your desired section.
